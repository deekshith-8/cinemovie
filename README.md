## Overview

CineMovie is a full-featured movie discovery app built with React and the TMDB API. Users can search films, filter by genre, explore detailed pages with cast, trailers and OTT availability, and manage a personal watchlist — all in a clean, responsive UI.

---

## Features

- Search movies with real-time results
- Filter by genre across the full TMDB catalog
- Detail pages with cast, trailers and OTT streaming providers
- Personal watchlist with persistent state
- Google OAuth and JWT-based authentication via MERN backend
- Fully responsive — works across all screen sizes

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React, Vite |
| Routing | React Router (HashRouter) |
| State Management | Context API |
| Data Source | TMDB API |
| Auth | JWT + Google OAuth (Passport.js) |
| Backend | Node.js, Express, MongoDB |
| Deployment | GitHub Pages |
